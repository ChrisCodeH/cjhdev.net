DualShield Assistant

DualShield Assistant is optional companion software to the DualShield Authentication Server, specifically made for environments where MySQL database replication is being used with DualShield. It provides system administrators with a monitoring interface to view the current status of the replication cluster as well as notifying by email if any network or replication errors occur. DualShield Assistant can also be used to clone an existing DualShield installation and setup a new replication cluster.

Deepnet CSR & Certificate Generator

The Deepnet CSR & Certificate Generator is a free tool which can be used to generate a Certificate Signing Request (CSR) to then submit to a Certificate Authority (CA) to obtain your Server Certificate. Additionally, after you have received your CA signed Certificate file, you can then use this tool to combine it with your original CSR's Private Key and any additional Intermediate CA Certificates to create a PKCS12 (.pfx) File. This ensures total confidentiality of your private key which may not be ensured by online CSR Generator's.

Card Scanner Rollout Tool to AD Users

I created this tool to solve an administration issue with the Deepnet DualShield system where it would be time consuming for administrators to rollout new access cards to a large number of users. In the tool, you can constantly search active directory users and scan access cards asynchronously to create a token import file for the DualShield Authentication System.
